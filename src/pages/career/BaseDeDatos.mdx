## Base De Datos

Despues de las hermosas clases de Arquitectura y Sistemas Operativos , volvimos a las clases de Rosalía con Base de datos , donde la primera clase ya fue una clase compleja de crear diagramas entidad Relación y una entrega para el fin de semana

### Clase I - 16 de agosto de 2024

<div class="w-full md:w-[800px] min-h-80 border bg-white shadow-lg p-6 mx-auto my-4 rounded-lg">
  <h3 class="text-3xl font-bold text-center mb-6 text-blue-600">Introducción</h3>
  <p class="text-lg text-gray-700 mb-4">
    En el entorno del mercado actual, la competitividad y la rapidez de maniobra de una empresa son imprescindibles para su éxito. 
    Para conseguirlo existe <span class="font-semibold">cada vez una mayor demanda de datos</span> y, por tanto, 
    <span class="font-semibold">más necesidad de gestionarlos</span>. Esta demanda siempre ha estado patente en empresas y sociedades, 
    pero en estos años se ha disparado debido al acceso multitudinario a las redes integradas en Internet y a la aparición de los 
    dispositivos móviles que también requieren esa información.
  </p>
  <p class="text-lg text-gray-700 mb-4">
    En informática se conoce como <span class="font-semibold">dato</span> a cualquier elemento informativo que tenga 
    <span class="font-semibold">relevancia para un usuario</span>. Desde su nacimiento, la informática se ha encargado de proporcionar 
    herramientas que faciliten la manipulación de los datos.
  </p>
  <p class="text-lg text-gray-700">
    Los sistemas de información actuales se basan en <span class="font-semibold">bases de datos (BD)</span> y 
    <span class="font-semibold">sistemas de bases de datos (SGBD)</span> que se han convertido en elementos imprescindibles de la 
    vida cotidiana de la sociedad moderna.
  </p>
</div>

<div class="w-full md:w-[800px] min-h-80 border bg-white shadow-lg p-6 mx-auto my-4 rounded-lg">
  <h3 class="text-3xl font-bold text-center mb-6 text-blue-600">Definición de Base de Datos</h3>
  <p class="text-lg text-gray-700 mb-4">
    Cada día, la mayoría de nosotros nos encontramos con actividades que requieren algún tipo de interacción con una base de datos 
    (ingreso en un banco, reserva de una entrada para el teatro, solicitud de una suscripción a una revista, compra de productos, ...).
  </p>
  <ul class="list-disc list-inside text-gray-700 mb-4">
    <li><span class="font-semibold">Una base de datos</span> se entenderá como una colección de datos relacionados entre sí y que tienen un significado implícito.</li>
    <li><span class="font-semibold">Por datos</span> queremos decir hechos conocidos que pueden registrarse y que tienen un significado implícito.</li>
  </ul>
  <p class="text-lg text-gray-700 mb-4">
    <span class="font-semibold">Ejemplo:</span> Una agenda con los nombres y teléfonos de un conjunto de personas conocidas es una base de datos, 
    puesto que es una colección de datos relacionados con un significado implícito.
  </p>
  <p class="text-lg text-gray-700 mb-4">
    La definición hace referencia a dos elementos para que un conjunto de datos constituya una Base de Datos:
  </p>
  <ol class="list-decimal list-inside text-gray-700 mb-4">
    <li><span class="font-semibold">Relaciones entre datos</span>, tema que se tratará en las secciones siguientes.</li>
    <li><span class="font-semibold">Significado implícito de los datos</span> que se atribuye dependiendo del contexto en que se utilizan los mismos.</li>
  </ol>
</div>

<div class="w-full md:w-[800px] min-h-80 border bg-white shadow-lg p-6 mx-auto my-4 rounded-lg">
  <h3 class="text-3xl font-bold text-center mb-6 text-blue-600">Evolución Histórica</h3>
  <p class="text-lg text-gray-700 mb-4">
    Los predecesores de los sistemas gestores de bases de datos fueron los sistemas gestores de ficheros o sistemas de archivos tradicionales.
  </p>
  <ol class="list-decimal list-inside text-gray-700 mb-4">
    <li><span class="font-semibold">Archivos tradicionales:</span> Consiste en almacenar los datos en archivos individuales, exclusivos para cada 
      <span class="font-semibold">aplicación particular</span>. En este sistema los datos pueden ser <span class="font-semibold">redundantes</span> 
      (repetidos innecesariamente) y la <span class="font-semibold">actualización</span> de los archivos es más lenta que en una base de datos.</li>
    <li><span class="font-semibold">Base de datos:</span> Es un <span class="font-semibold">almacenamiento</span> de datos formalmente definido, 
      <span class="font-semibold">controlado centralmente</span> para intentar servir a múltiples y diferentes aplicaciones. La base de datos es 
      una fuente de datos que son compartidos por numerosos usuarios para diversas aplicaciones.</li>
  </ol>
</div>

<div class="w-full md:w-[800px] min-h-80 border bg-white shadow-lg p-6 mx-auto my-4 rounded-lg">
  <h3 class="text-3xl font-bold text-center mb-6 text-blue-600">Evolución de los Tipos de Base de Datos</h3>
  <p class="text-lg text-gray-700 mb-4">
    Concidiendo con la evolución histórica de las bases de datos, estas han utilizado distintos modelos:
  </p>
  <ul class="list-disc list-inside text-gray-700 mb-4">
    <li>Jerárquicos</li>
    <li>En red</li>
    <li>Relacionales <span class="font-semibold">(Lo que vamos a estudiar)</span></li>
    <li>Multidimensionales</li>
    <li>De objetos</li>
  </ul>
  <h4 class="text-2xl font-bold text-center mb-4 text-blue-600">Evaluación de los Sistemas de Base de Datos</h4>
  <p class="text-lg text-gray-700 mb-4">
    Inicio: parte de la necesidad de almacenar grandes cantidades de datos.
  </p>
  <ol class="list-decimal list-inside text-gray-700 mb-4">
    <li><span class="font-semibold">Década de 1950:</span> Da origen a las cintas magnéticas.</li>
    <li><span class="font-semibold">Década de 1960:</span> Implementación de los discos.</li>
    <li><span class="font-semibold">Década de 1970:</span> Nacen las bases de datos relacionales y Oracle.</li>
    <li><span class="font-semibold">Década de 1980:</span> El modelo relacional se posiciona en el mercado de las bases de datos.</li>
    <li><span class="font-semibold">Actualidad:</span> Diversidad de modelos y tecnologías.</li>
  </ol>
</div>
